# Rain

A hobby build system and language.

- [Example Scripts](https://github.com/RuairidhWilliamson/rain/tree/main/core/tests/scripts)

### Installation

Prebuilt releases are available:

- [Windows x64](https://storage.googleapis.com/rain-e870e063/releases/0.10.0/rain-windows-amd64.exe)
- [Linux x64](https://storage.googleapis.com/rain-e870e063/releases/0.10.0/rain-linux-amd64)

or install from source

```bash
git clone git@github.com:RuairidhWilliamson/rain.git
cd rain
cargo rain exec install_cli
```

### Usage

```bash
rain --help
```

### Development

```bash
cargo rain check
```
