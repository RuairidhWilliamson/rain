# Rain

### Installation

Prebuilt releases are available:

- [Windows x64](https://storage.googleapis.com/rain-e870e063/releases/0.6.0/rain-windows-amd64.exe)
- [Linux x64](https://storage.googleapis.com/rain-e870e063/releases/0.6.0/rain-linux-amd64)

```bash
git clone git@github.com:RuairidhWilliamson/rain.git
cd rain
cargo install --path cli --locked
```

### Usage

```bash
rain --help
```

### Development

```bash
cargo rain check
```
